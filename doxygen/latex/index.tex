 A library adding an easy way to do scenes for raylib(or really anything)!

Supports single header and header/implementation modes\+:


\begin{DoxyItemize}
\item For single header just copy {\ttfamily \mbox{\hyperlink{rayscenes_8hpp_source}{rayscenes.\+hpp}}} from the root dir and include it
\begin{DoxyItemize}
\item To use the implementation, define {\ttfamily RAYSCENE\+\_\+\+IMPLEMENTATION} before including {\ttfamily \mbox{\hyperlink{rayscenes_8hpp_source}{rayscenes.\+hpp}}}
\end{DoxyItemize}
\item For header/implementation separated, copy {\ttfamily \mbox{\hyperlink{rayscenes_8hpp_source}{include/rayscenes.\+hpp}}} and {\ttfamily src/rayscenes.\+cpp} from the root dir and include them
\end{DoxyItemize}

Written in for C++20, should support C++17.
